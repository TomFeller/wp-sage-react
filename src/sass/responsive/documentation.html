Responsive Mixins

The responsive mixins file is used to build responsive css elements.
There are 3 mixins you can include (listed in the table below).
All mixins needs to get different kind of parameters. These are the parameters that can used:

mobile | mobile-lg | tablet | desktop | desktop-lg | wide

The parameter that will send define witch query to use (with min-width on the query).
The viewport variables is in the '_responsive-variable.scss' file.

Available Mixins:

name Description Example
viewport-size Use to add css to a specific query.
The mixin can get one of the available queries or a custom number of pixels. 
    
        .item {
            @include viewport-size(tablet) {
                // That code will appear in min-width of $viewport-tablet variable.
            }
        }
responsive-prop Use when need to give different values in every query for specific css property. 
    
        .item {
            @include responsive-prop(width, 10px, 20px, 30px, 40px, 50px ,60px, 70px) {
                // all '.item' class will get the width property in the query sizes respectively.
            }
        }
responsive-mtd Use to add css only for mobile, tablet, and desktop. 
    
        .item {
            @include responsive-mtd(width, 10px, 20px, 30px) {
                // query in mobile tablet and desktop respectively.
            }
        }

Live example:

SCSS CSS 
    
    .wrapper {
        @include viewport-size(tablet) {
            background-color: red;
        }
    }
    

    
    @media only screen and (min-width: 768px) {
        .wrapper {
            background-color: red; }
        }
    }
    

    
    .wrapper {
        @include viewport-size(400) {
            background-color: blue;
        }
    }
    

    
    @media only screen and (min-width: 400px) {
        .wrapper {
            background-color: blue;
        }
    }
    

    
    .wrapper {
        @include responsive-mtd('width', 100px, 200px, 300px);
    }
    

    
    .wrapper {
        width: 100px;
    }

    @media only screen and (min-width: 768px) {
        .wrapper {
            width: 200px;
        }
    }

    @media only screen and (min-width: 992px) {
        .wrapper {
            width: 300px;
        }
    }
